Welcome to the **Tim's Clips** web application, a premium platform for booking barber appointments, showcasing real client results, and managing the business via a custom admin dashboard.

## Features

- **Modern & Premium UI:** Designed carefully to provide a high-end luxury feel.
- **Client Booking:** Easy, seamless appointment booking for clients.
- **Admin Dashboard:** A secured area for the shop owner to view, manage, and modify bookings.
- **Portfolio Gallery:** Display high-quality images of past haircuts.
- **Firebase Integration:** Real-time data storage, and authentication mapping (tracking admin users).

## Technologies Used

- React (Vite)
- Tailwind CSS & Framer Motion for styling and animations
- Firebase (Authentication, Firestore)

## Local Development

**Prerequisites:** Node.js installed

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Firebase Configuration:**
   Ensure the `firebase-applet-config.json` file is correctly formatted with your Firebase project's credentials. This configures the Firebase client application.

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. **Building for Production:**
   ```bash
   npm run build
   ```
